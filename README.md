# Protractor Test Framework
This is a simple protractor testing framework with JavaScript, it contains some quality of life implementations as task runners, screenshot on failure and html reporting.

## Tools
These are the tools ultilized in this project

* [Jasmine](https://jasmine.github.io/)
* [Protractor](https://www.protractortest.org/)
* [Selenium Webdriver](https://www.selenium.dev/documentation/en/)
* [Jasmine-spec-reporter](https://www.npmjs.com/package/jasmine-spec-reporter)
* [Protractor Style Guide](https://github.com/CarmenPopoviciu/protractor-styleguide)
* [Jasmine2-html-reporter](https://www.npmjs.com/package/protractor-jasmine2-html-reporter)

## Prerequisites
First things first you need [Node.js](https://nodejs.org/en/) installed, then you just need to run the code below:
```bash
npm i / npm install
```
This command will install all project dependencies that are into package.json file.

Then you will need to update you webdriver, it already comes with protractor but it may not be the most recent version of [Selenium Webdriver](https://www.selenium.dev/documentation/en/).
```bash
webdriver-manager update / npm run wd:update
```

# Be Aware
You need your Google Chrome or Firefox up-to-date to get this project up and running flawlessly
